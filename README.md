## 🚀 김수빈 Portfolio

> **화면 너머의 데이터 흐름을 이해하고, 기획부터 배포까지 팀과 긴밀하게 소통하며 서비스를 완성하는 풀스택 개발자 김수빈입니다.**

이전 사회생활에서 인사총무 및 비서 직무를 수행하며 다양한 시스템을 직접 사용하는 '사용자'의 입장을 깊이 있게 경험했습니다.

프로젝트를 진행하며 "어떤 데이터 흐름이 개선되어야 서비스가 더 안정적이고 편리해질까?"를 치열하게 고민했고, 

이는 단편적인 기능 구현을 넘어 **서비스 전체 아키텍처 흐름을 탐구하는 개발자**로서의 가치관으로 이어졌습니다.

또한 과거 직무를 통해 체득한 **조직 커뮤니케이션 역량과 꼼꼼한 관리 능력**을 바탕으로 다음과 같은 핵심 역할을 주도했습니다:

- 요구사항 정의 및 기획 리드: 복잡하게 얽힌 비즈니스 요구사항을 구조적으로 명확히 명세화하여 팀의 개발 방향성을 잡았습니다.
- 소통의 가교 역할: 팀원들이 하나의 목표를 향해 유기적으로 협업할 수 있도록 팀원 간 의견을 조율했습니다.
- 서비스 전체 QA 및 품질 검증 담당: 사용자의 시선과 개발자의 시선을 동시에 유지하며 꼼꼼한 테스트 프로세스를 구축했고, 숨은 오류들을 선제적으로 찾아내 서비스의 완성도와 안정성을 극대화했습니다.

단순히 코드만 짜는 것을 넘어, 사용자의 불편함을 기술로 해결하고 팀의 생산성을 높이는 '함께 일하고 싶은 든든한 동료'가 되겠습니다.
